# Mouse Dynamics Authentication with Explainable AI (XAI)

This repository contains a complete machine learning pipeline for user authentication based on mouse movement biometrics. The project focuses on extracting behavioral features from raw mouse event data and providing transparent decision-making using Explainable AI (XAI) techniques.

## Project Overview

The goal of this thesis is to identify users based on their unique way of interacting with a computer mouse. By analyzing spatial and temporal characteristics, the system builds a biometric profile for each user.
